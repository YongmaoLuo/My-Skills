# Skill Creating

A comprehensive guide for creating Agent Skills following the Agent Skills open standard.

## What This Skill Does

This skill provides complete instructions, examples, and best practices for creating Agent Skills that can be used across multiple AI agent platforms including:
- Claude Code
- Cursor
- OpenCode
- Claude.ai
- And other Agent Skills-compatible agents

## When to Use This Skill

Use this skill when you:
- Want to create a new Agent Skill from scratch
- Need to understand the Agent Skills format specification
- Are troubleshooting skill creation issues
- Want to validate an existing skill
- Need examples of skill patterns and best practices

## What You'll Learn

This skill covers:
- Agent Skills directory structure
- SKILL.md format and frontmatter requirements
- Progressive disclosure principles
- Optional directories (scripts/, references/, assets/)
- Naming conventions and validation
- Best practices for skill design
- Platform-specific installation instructions
- Publishing and distribution

## Skill Structure

```
skill-creating/
├── SKILL.md                          # Main skill instructions
├── references/

```
